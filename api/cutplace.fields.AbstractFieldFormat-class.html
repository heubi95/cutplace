Abstract format description of a field in a data file to validate which acts as base for all other field formats. To implement another field format, it is usually sufficient to:

  1. Overload __init__() but call super(..., self).__init__(...) from it.
  2. Implement validatedValue().

validatedValue(self, value)

source code 

The value in its native type for this field.

By the time this is called it is already ensured that:

  • value is not an empty string
  • value contains only valid characters
  • trailing blanks have been removed from value for fixed format data

Concrete fields formats must override this because the default implementation just raises a NotImplementedError.
