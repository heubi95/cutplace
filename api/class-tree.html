Class Hierarchy

  • object: The most base type
    • cutplace.checks.AbstractCheck: Abstract check to be used as base class for other checks.
      • cutplace.checks.DistinctCountCheck: Check to ensure that the number of different values in a field matches an expression.
      • cutplace.checks.IsUniqueCheck: Check to ensure that all rows are unique concerning certain key fields.
    • cutplace.fields.AbstractFieldFormat: Abstract format description of a field in a data file to validate which acts as base for all other field formats.
      • cutplace.fields.ChoiceFieldFormat: Field format accepting only values from a pool of choices.
      • cutplace.fields.DateTimeFieldFormat: Field format accepting values that represent dates or times.
      • cutplace.fields.DecimalFieldFormat: Field format accepting decimal numeric values, taking the data format properties data.KEY_DECIMAL_SEPARATOR and data.KEY_THOUSANDS_SEPARATOR into account.
      • cutplace.fields.IntegerFieldFormat: Field format accepting numeric integer values with fractional part.
      • cutplace.fields.PatternFieldFormat: Field format accepting values that match a pattern using "*" and "?" as place holders.
      • cutplace.fields.RegExFieldFormat: Field format accepting values that match a specified regular expression.
      • cutplace.fields.TextFieldFormat: Field format accepting any text.
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • cutplace.tools._BaseCutplaceError: Exception that supports a message describing the error and an optional location in the input where the error happened.
          • cutplace.tools.CutplaceError: Error detected by cutplace caused by issues in the ICD or data.
            • cutplace.checks.CheckError: Error to be raised when a check fails.
            • cutplace.checks.CheckSyntaxError: Error to be raised when the specification of check in the ICD is broken.
            • cutplace.sniff.CutplaceSniffError: Error to indicate that the format of some content could not be sniffed.
            • cutplace.data.DataFormatSyntaxError: Error in data format declaration.
            • cutplace.data.DataFormatValueError: Error in data caused by violating the data format.
            • cutplace.fields.FieldLookupError: Error raised when a field cannot be found.
            • cutplace.fields.FieldSyntaxError: Error raised when a field definition in the ICD is broken.
            • cutplace.fields.FieldValueError: Error raised when AbstractFieldFormat.validated detects an error.
            • cutplace.interface.IcdSyntaxError: General syntax error in the specification of the ICD.
            • cutplace.ranges.RangeSyntaxError: Error in Range declaration.
            • cutplace.ranges.RangeValueError: Error raised when ranges.validate() detects that a value is outside the expected ranges.
          • cutplace.tools.CutplaceUnicodeError: Error detected by cutplace caused by improperly encoded ICD or data.
    • cutplace.interface.BaseValidationListener: Listener to process events during InterfaceControlDocument.validate().
    • cutplace.tools.ErrorInfo: Container for sys.exc_info() that also can reraise() an existing exception while preserving the stack trace.
    • cutplace.tools.InputLocation: Location in an input file, consisting of line, an optional column (pointing at a single character) and an optional cell (pointing a cell in a structured input such as CSV).
    • cutplace.interface.InterfaceControlDocument: Model of the data driven parts of an Interface Control Document (ICD).
    • cutplace.ranges.Range: A range that can be used to validate that a value is within it.
    • cutplace.interface.Validator: Validator for tabular data organized in rows described by an InterfaceControlDocument.
    • cutplace.interface.Writer: Writer to write data to a filelike output while validating that they conform to an InterfaceControlDocument.
    • cutplace.data._BaseDataFormat: Data format acting as base for other data formats.
      • cutplace.data._BaseSpreadsheetDataFormat: Base data format for spreadsheet formats.
        • cutplace.data.ExcelDataFormat: Data format for Excel spreadsheets.
        • cutplace.data.OdsDataFormat: Data format for ODS as created by OpenOffice.org's Calc.
      • cutplace.data._BaseTextDataFormat: Base data format that supports an "encoding" and "line delimiter" property.
        • cutplace.data.DelimitedDataFormat: Data format for delimited data such as CSV.
          • cutplace.data.CsvDataFormat: Data format for comma separated values (CSV).
        • cutplace.data.FixedDataFormat: Data format for fixed data.
    • cutplace.sniff._ColumnSniffInfo
