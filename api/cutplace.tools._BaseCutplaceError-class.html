source code 
Create exception that supports a message describing the error and an optional InputLocation in the input where the error happened. If the message is related to another location (for example when attempting to redefine a field with the same name), seeAlsoMessage should describe the meaning of the other location and seeAlsoLocation should point to the location. If the exception is the result of another exception that happened earlier (for example a UnicodeError, cause should contain this exception to simplify debugging.
