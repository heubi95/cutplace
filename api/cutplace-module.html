This is the API documentation for cutplace, a tool to validate that tabular and flat data conform to an interface control document (ICD).

For more information about cutplace, visit https://github.com/roskakori/cutplace.

Version: 0.7.1

Submodules [hide private]
  • cutplace.checks: Standard checks that can cover a whole row or data set.
  • cutplace.data: Data formats that describe the general structure of the data.
  • cutplace.fields: Standard field formats supported by cutplace.
  • cutplace.interface: Interface control document (ICD) describing all aspects of a data driven interface.
  • cutplace.ranges: Ranges check if certain values are within it.
  • cutplace.sniff: Heuristic data analysis to figure out file types and field formats.
  • cutplace.tools: Various utility classes and functions.
